Instrucciones

Dado Mentiroso (también conocido como Dudo, Perudo o Cacho) es un juego de dados y faroles para 2 o más jugadores.

Objetivo: Ser el último jugador con dados.

Cómo se juega:

  • Cada jugador tiene 5 dados y un cubilete.
  • Todos agitan sus dados y miran su resultado en secreto.
  • Por turnos, se hacen apuestas sobre el total de dados con un cierto valor entre TODOS los jugadores. Ej: "tres 5s".
  • Cada apuesta debe superar a la anterior: más dados del mismo valor (3 cinco → 4 cinco), el mismo número de dados con un valor mayor (3 cinco → 3 seis), o más dados de cualquier valor.
  • El siguiente jugador puede subir la apuesta, decir "Dudo" o decir "Exacto".
  • Dudo: Se revelan todos los dados. Si la apuesta era correcta (hay al menos esa cantidad), el que dijo "Dudo" pierde un dado. Si era falsa (hay menos), el apostante pierde un dado.
  • Exacto: Se revelan todos los dados. Si hay exactamente esa cantidad, todos los demás pierden un dado y el que dijo "Exacto" recupera uno si le falta. Si no es exacto, el que dijo "Exacto" pierde un dado.

En esta app:

  • Tirar: Agita el móvil o pulsa el botón TIRAR.
  • -1 Dado: Cuando pierdas una ronda.
  • Reset: Vuelve a empezar.
